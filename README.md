# The_Creative_Yumland_Project
README — The Wonders of Svaneti
Projet Creative-Yumland • CY Tech préING2 • 2025-2026

**C'est quoi ce projet ?**
- C'est le site web d'un restaurant géorgien appelé The Wonders of Svaneti. Le but final sur tout le semestre c'est d'avoir une application complète qui gère la chaîne d'une commande — de la consultation de la carte jusqu'à la livraison.

**Phase 1 — Retour**
- On a créé 9 pages statiques qui couvrent tous les profils utilisateurs du projet :
- Page: Profil(Client) / home_page.html(Client) / presentation_page.html(Client)/ inscription.html(Client) / connexion.html(Client) / profil.html(Client) /notation.html(Client) / admin.html(Administrateur) / commandes.html(Restaurateur) / livraison.html(Livreur)
- Tout le CSS est centralisé dans un seul fichier : restaurant.css.

**Stack technique**
- Phase 1: HTML5/CSS3
- Phase 2: PHP
- Phase 3: JS/Async

**Lancer le projet en 5 étapes** 
- Telecharger de préférence le dossier avec la commande: "git clone https://github.com/Ladino21/The_Creative_Yumland_Project The_Wonders_of_Svaneti" !!
- Telecharger le logiciel "xampp" à l'adresse suivante "https://www.apachefriends.org/download.html" selon votre système d'exploitation pour avoir le retour visuel escompter
- Dans xampp, lancez "Apache" le serveur web
- Placez votre dossier "The_Wonders_of_Svaneti" dans le dossier "xampp"
- Entrez l'URL suivant dans le moteur de recherche de votre choix: "http://localhost/Projet_restaurant/home_page.php" !!
